Всем привет!

Данный тестовый проект позволяет:
1. Всем пользователям:
    1.1 Просматривать публикации на вкладке Список публикаций
    1.2 Осуществлять поиск по тексту публикаций на вкладке Список публикаций
    1.3 Пройти регистрацию на вкладке Регистрация
          Для регистрации необходимо
          1.3.1 Логин — латинские буквы, цифры, спецсимволы
          1.3.2 Email в формате xxxx@xxx.xxx
          1.3.3 Пароль не менее 4ех символов
2. Зарегистрированным пользователям:
    2.1 Выполнять функции перечисленные в пунктах 1.1 и 1.2
    2.2 Добавлять свои публикации на вкладке Добавить публикацию
    2.3 Осуществлять Вход/Выход, используя свой email в качестве логина и пароль
3. Зарегистрированным пользователям с правами администратора (используйте для регистрации/входа email – admin@mail.com, пароль — 1234):
    3.1 Выполнять функции перечисленные в пунктах 1.1, 1.2, 2.2, 2.3
    3.2 Удалять чужие публикации на вкладке Редактировать список публикаций
    3.3 Просматривать список зарегистрированных пользователей на вкладке Зарегистрированные пользователи

Стек технологий: Java, Spring boot, Spring Security, Spring Data JPA, Postgres, Thymeleaf, Bootstrap, HTML.
