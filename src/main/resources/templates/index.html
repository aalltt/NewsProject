    Всем привет!

    Этот тестовый проект позволяет:
  • Добавлять свои публикации, сохраняя их в БД
  • Просматривать публикации
  • Удалять публикации, если авторизоваться с учетной записью администратора (логин - Bob, пароль - 1234)


Стек технологий: Java, Spring boot, Spring Security, Spring Data JPA, Postgres, Thymeleaf, Bootstrap, HTML
